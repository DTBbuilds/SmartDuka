## Interactions

### Hover States
- Quick Add Card: Shadow increases
- Products Card: Slight scale up
- Categories Card: Slight scale up
- Bulk Ops Card: Shadow increases

### Click States
- Quick Add: Expands/collapses
- Bulk Ops: Opens CSV import modal
- Products/Categories: No action (display only)

### Focus States
- All interactive elements: Blue outline
- Buttons: Keyboard focus visible
- Form inputs: Blue border
