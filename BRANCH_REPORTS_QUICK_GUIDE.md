# Branch Reports & Stock Sharing - Quick Guide

**Date**: November 11, 2025

---

## Access Reports

1. Go to Admin Dashboard
2. Click "Reports & Analytics" (or navigate to /admin/reports)
3. Choose tab: "Branch Performance" or "Stock Sharing"

---

## Branch Performance Report

### View Performance
1. Select branch from dropdown
2. Choose time period (Today/Week/Month)
3. View performance metrics

### Metrics Displayed
- **Total Revenue**: Branch sales amount with trend
- **Total Orders**: Number of transactions
- **Average Order Value**: Revenue per transaction
- **Inventory Status**: Total items and alerts
- **Staff Performance**: Staff count and sales per staff
- **Performance Score**: 0-100 rating with emoji stars

### Export Report
1. Click "Export" button
2. CSV file downloads with all metrics
3. File named: branch-report-[name]-[date].csv

### Performance Ratings
- ⭐⭐⭐⭐⭐ Excellent (90-100)
- ⭐⭐⭐⭐ Good (75-89)
- ⭐⭐⭐ Fair (60-74)
- ⭐⭐ Poor (<60)

## Stock Sharing

### Create Transfer
1. Select "From Branch" (source)
2. Select "To Branch" (destination)
3. Choose product
4. Enter quantity (max = available stock)
5. Click "Create Transfer"

### Manage Transfers
- **Pending**: Shows "Approve" and "Reject" buttons
- **Approved**: Transfer approved, awaiting completion
- **Completed**: Transfer finished
- **Rejected**: Transfer cancelled

### Transfer Status
- ⏳ Pending: Awaiting approval
- ✅ Approved: Approved, in progress
- ✓ Completed: Successfully transferred
- ❌ Rejected: Cancelled

### Approve Transfer
1. Find pending transfer
2. Click "Approve" button
3. Transfer moves to approved status

### Reject Transfer
1. Find pending transfer
2. Click "Reject" button
3. Transfer cancelled

## Common Tasks

### Check Branch Sales Performance
1. Go to Branch Performance tab
2. Select branch
3. Choose "This Month"
4. View revenue and order metrics
5. Check performance score

### Identify Low Stock Branches
1. Go to Branch Performance tab
2. Review "Inventory Status" card
3. Check "Low Stock Items" count
4. Note branches with alerts

### Transfer Stock to Low Stock Branch
1. Go to Stock Sharing tab
2. Select high-stock branch as "From"
3. Select low-stock branch as "To"
4. Choose product
5. Enter quantity
6. Click "Create Transfer"
7. Approve transfer

### View Transfer History
1. Go to Stock Sharing tab
2. Scroll to "Stock Transfers" section
3. See all transfers with status
4. View dates and quantities

### Optimize Stock Distribution
1. Review performance reports for each branch
2. Identify branches with excess stock
3. Identify branches with low stock
4. Create transfers to balance inventory
5. Monitor turnover rates

## Tips & Best Practices

### Performance Analysis
- Check weekly reports to identify trends
- Compare branches to find best performers
- Use top product data for marketing
- Monitor staff sales per capita

### Stock Management
- Transfer slow-moving items to branches with demand
- Balance inventory to reduce holding costs
- Address low-stock alerts before stockouts
- Monitor transfer history for patterns

### Optimization
- Use performance scores to identify issues
- Transfer inventory based on demand patterns
- Keep inventory turnover above 80%
- Monitor staff productivity metrics

## Troubleshooting

### Issue: Branch not showing in dropdown
**Solution**: Ensure branch is created and active

### Issue: No performance data
**Solution**: Check that branch has sales data for selected period

### Issue: Cannot create transfer
**Solution**: Verify source branch has sufficient stock

### Issue: Transfer not appearing
**Solution**: Refresh page or check transfer status

### Issue: Export not working
**Solution**: Check browser download settings
